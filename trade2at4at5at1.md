## 2. 实现

| 项目 | trade2at4 / 原 at5 | trade2at4at5（本版） |
|------|---------------------|----------------------|
| `get_market_score` 数据源 | `000001.XSHG` 上证指数 | **`000300.XSHG` 沪深300** |
| `set_benchmark` | `000300.XSHG` | 不变（已一致） |
| 参数 | — | `g.market_index = '000300.XSHG'` |

### 计分逻辑（不变）

```
近60日收盘 → MA20 / MA60
收盘 > MA20 > MA60  → 80分
收盘 > MA20         → 60分
收盘 > MA60         → 40分
否则                → 30分
20日涨幅 > 5%       → +10（上限90）
20日跌幅 > 5%       → -10（下限20）
```

### 用途（不变）

- 买入：`market_score < 30` 停买  
- 止损：`market_score < 40` 固定止损 8%→6%  
- 仓位：`calculate_position_ratio` 分档
