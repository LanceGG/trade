# ta1 回撤与止损策略分析

> 源码：`ta1`（trade2at4at5at4 基线）  
> 持仓上限 5 只 | 买入门槛大盘分 ≥ 30 | 卖出在 10:00 / 14:30 复检  
> 实证对照：`ta2log.txt`（ta2 = ta1 卖出逻辑 + 分析日志，2018-11 ~ 2026-06）

---

## 1. 风控架构总览

ta1 对单股的「回撤/止损」分为 **五条独立链路**，无统一风控中枢，按固定优先级在 10:00 串行判断：

```
                    ┌──────────────────────────────────────┐
                    │           单股持仓风控（ta1）            │
                    └──────────────────────────────────────┘
                                      │
     ┌────────────┬────────────┬─────┴─────┬────────────┬────────────┐
     ▼            ▼            ▼           ▼            ▼            ▼
 ①亏损止损    ②盈利回撤     ③通道3日     ④双死叉      ⑤选股回撤     ⑥组合仓位
 (成本线)    (高点线)     (趋势通道)   (技术信号)   (入场过滤)   (大盘分)
 10:00/14:30  10:00/14:30  仅10:00    仅10:00     07:30选股    约束新买
```

| 链路 | 类型 | 参照基准 | 触发时点 | 动作 |
|------|------|----------|----------|------|
| ① ATR/固定止损 | 亏损侧 | **买入成本** | 10:00、14:30 | 清仓 |
| ② 盈利回撤止盈 | 盈利侧 | **持仓最高价** | 10:00、14:30 | 减半 / 清仓 |
| ③ 通道 3 日 | 趋势侧 | **多空通道上轨下沿** | 盘后计数 → 次日 10:00 | 清仓 |
| ④ BBI+MACD 双死叉 | 技术侧 | **昨收 vs 昨指标** | 10:00 | 清仓 |
| ⑤ 选股回撤过滤 | 预防侧 | 52 周高点 / MA200 | 07:30 选股 | 不入池 |
| ⑥ 大盘分仓位 | 组合侧 | 上证趋势 | 10:00 买入 | 限制暴露，**不减仓** |

**未覆盖：** 无个股买入冷静期、无行业互斥、无单一 MACD/BBI 减半、无 MA20 减半、无浮盈 20%/35% 阶梯止盈。

## 2. 参数一览

```python
# 亏损止损
g.hard_stop = 0.08                    # 固定止损底线 8%
g.hard_stop_tight = 0.06              # 弱市收紧至 6%
g.hard_stop_score_threshold = 40      # 大盘分 < 40 触发弱市止损
g.atr_stop_mult = 2.0                 # ATR 放宽倍数

# 盈利回撤（at4）
g.trail_profit_threshold = 0.50       # 浮盈 > 50% 才启用
g.trail_stop_half = 0.08              # 自高点回撤 8% → 减半
g.trail_stop_full = 0.12              # 自高点回撤 12% → 清仓

# 选股回撤
g.max_correction_from_high = 0.30     # 距 52 周高点回撤不超过 30%

# 状态
g.position_high = {}                  # 持仓以来最高价
g.trail_half_done = {}                # 是否已减半
g.break_upper_count = {}              # 连续跌破通道天数
```

## 3. ① 亏损止损（成本回撤）

### 3.1 公式

```text
pnl_ratio = (现价 - 成本) / 成本

threshold = max( hard_stop_eff , 2 × ATR14 / 收盘价 )

hard_stop_eff = min(8%, 6%)  当大盘分 < 40
              = 8%            否则
```

实现：`_get_stop_threshold()` → `_check_stop_loss()` / `market_open_trade` 2.1 节。

### 3.2 行为特征

| 维度 | 说明 |
|------|------|
| 参照价 | `position.avg_cost`，**非**买入后最高价 |
| ATR 作用 | 高波动股止损**放宽**（取 max），熊市 6% 底线可被 2×ATR 覆盖到 12%~20% |
| 检查频率 | **10:00** + **14:30**（盘中急跌可下午触发） |
| 触发结果 | 清仓 + `_cleanup_position_state` |

### 3.3 实证（ta2log，45 笔 ATR 止损）

- 胜率 **0%**，均亏 **-11.9%**
- 81% 持仓 ≤ 15 天
- 典型场景：急跌日 10:00 未触及，14:30 或次日触发；ATR 放大后实际亏损常超 -8%

### 3.4 设计评价

- **优点**：弱市收紧 6%，有基本风控
- **缺口**：ATR 放宽在暴跌股上反而**延迟**止损；无跳空专用处理；止损后无冷静期，可立即买回

## 4. ② 盈利回撤止盈（高点回撤 Trailing Stop）

### 4.1 公式

```text
启用条件：pnl_ratio > 50%（相对成本）

dd_from_peak = (现价 - position_high) / position_high

dd ≤ -12%  → 清仓
dd ≤ -8% 且未减半 → 减仓 50%，标记 trail_half_done=True
```

`position_high` 在 10:00 巡检、14:30 巡检、**盘后收盘**更新（`after_market_close`）。

### 4.2 行为特征

| 维度 | 说明 |
|------|------|
| 门槛 | 浮盈须 **>50%** 才启动，30%~50% 盈利段**无**移动保护 |
| 减半 | 每只股票**一生仅减半一次**（`trail_half_done`） |
| 减半后 | 剩余仓位仍受 12% 高点回撤清仓约束 |
| 与止损关系 | 先检查亏损止损，再检查盈利回撤；浮盈股通常不触发 8% 成本止损 |

### 4.3 实证（ta2log）

| 类型 | 笔数 | 均盈亏 | 说明 |
|------|------|--------|------|
| 盈利回撤**清仓** | 31 | **+106.5%** | 利润主引擎 |
| 盈利回撤**减半** | 32 | 触发时均浮盈 +94% | 多数随后 12% 回撤清仓 |

### 4.4 设计评价

- **优点**：大牛股利润锁定有效（31 笔清仓全盈）
- **缺口**：50% 门槛过高，中等盈利（20%~50%）仅靠通道/死叉保护；8%/12% 回撤阈值在震荡市可能偏紧

## 5. ③ 通道 3 日跌破（趋势回撤）

### 5.1 指标计算

`calc_dktd_channel()`：量价加权 EXPMA 合成 CTA2，**上轨下沿 = CTA2 × 1.01**。

### 5.2 计数与触发

| 时点 | 动作 |
|------|------|
| **15:00 盘后** | 若收盘 < 上轨下沿 → `break_upper_count += 1`；否则计数清零 |
| **10:00 次日** | 若 `break_upper_count >= 3` → **清仓** |

**仅 10:00 卖出**，14:30 不检查通道。

### 5.3 行为特征

- 与盈亏无关：微利、亏损均可触发
- 连续 3 **交易日**收盘跌破，非盘中瞬时
- 洗盘时易累计 1~2 天预警后第 3 天清仓

### 5.4 实证（ta2log）

| 指标 | 数值 |
|------|------|
| 清仓笔数 | **271**（占卖出 60%） |
| 胜率 | 35.8% |
| 均盈亏 | +1.4%（结构贡献接近 0） |
| 亏损子集 | 172 笔，均亏 -4.6%，均持仓 17 天 |

**通道 3 日是换手与噪声交易最大来源。**

## 6. ④ BBI + MACD 双死叉

### 6.1 判定（均基于**昨日**数据）

```text
BBI = (MA6 + MA12 + MA24 + MA48) / 4
BBI死叉：昨收 < 昨BBI 且 前日收 >= 前日BBI

MACD(5,10,15) 死叉：昨 MACD < 昨 Signal 且 前日 MACD >= 前日 Signal

双死叉：两者同时为真 → 10:00 清仓
```

### 6.2 行为特征

| 维度 | 说明 |
|------|------|
| 单一死叉 | **不卖出**（ta1 无减半） |
| 与浮盈关系 | **无分级**：盈利 5% 与亏损 5% 同等对待 |
| 检查时点 | 仅 **10:00** |
| 数据 | `attribute_history` 昨收序列，10:00 用昨日收盘判断 |

### 6.3 实证（ta2log）

| 指标 | 数值 |
|------|------|
| 笔数 | 101 |
| 胜率 | **10.9%** |
| 均盈亏 | **-3.4%** |
| ≤15 天占比 | **90%** |

典型「刚买就死叉」震荡洗出形态。

## 7. ⑤ 入场侧回撤过滤（非持仓止损）

在 `get_canslim_pool()` 技术面过滤，**防止买入深度回撤股**：

| 规则 | 条件 | 参数 |
|------|------|------|
| 52 周高点回撤 | `现价 ≥ 52周高 × (1 - 30%)` | `max_correction_from_high=0.30` |
| MA200 趋势 | `现价 ≥ MA200 × 0.95` | 硬编码 0.95 |
| Stage2 前提 | `detect_cycle_stage` 内再次确认 MA200 | 同上 |

**与持仓止损独立**：已持仓股跌破 MA200 **不会**因此卖出。

## 8. ⑥ 组合层：大盘分与仓位（非个股止损）

### 8.1 大盘分 `get_market_score()`

上证 60 日线：MA20/MA60 位置 + 20 日涨跌幅调整 → 20~90 分。

### 8.2 仓位映射 `calculate_position_ratio()`

| 大盘分 | 目标仓位 |
|--------|----------|
| ≥ 70 | 100% |
| ≥ 55 | 80% |
| ≥ 40 | 60% |
| ≥ 25 | 40% |
| < 25 | 20% |

### 8.3 实际作用

- **仅约束 10:00 新买单票目标市值** = `总资产 × position_ratio / 5`
- **不会**因大盘走弱主动削减已有持仓
- 买入暂停：大盘分 **< 30** 停止新开仓
- 止损收紧：大盘分 **< 40** 时 `hard_stop` 6%（见 ①）

## 9. 日内执行时间线

```
07:30  before_trading_start
         └─ 选股（含 52周/MA200 回撤过滤）
         └─ g.sold_today 清空

10:00  market_open_trade  【主卖出窗口】
         └─ 对每只持仓，按序：
              2.1 亏损止损 → 清仓
              2.2 盈利回撤 → 减半/清仓
              2.3 通道≥3天 → 清仓
              2.4 双死叉 → 清仓
         └─ 买入（大盘分≥30，按仓位预算）

14:30  afternoon_stop_check  【复检窗口】
         └─ 盈利回撤（减半/清仓）
         └─ 亏损止损
         └─ ✗ 不检查通道、不检查死叉

15:00  after_market_close
         └─ 更新 position_high（收盘价）
         └─ 更新 break_upper_count（通道计数）
         └─ 组合净值 / 最大回撤统计
```

### 10:00 卖出优先级（严格串行，先触发先执行）

```
止损(①) > 盈利回撤(②) > 通道3日(③) > 双死叉(④)
```

## 11. 机制交互与结构性问题

### 11.1 短持洗出 vs 长持盈利

| 持仓天数（ta2log 清仓） | 胜率 | 均盈亏 |
|-------------------------|------|--------|
| ≤ 15 天 | ~6% | **-5.5%** |
| > 60 天 | 100% | **+74.5%** |

通道 3 日、双死叉、ATR 止损合计贡献大量 **≤15 天** 卖出，与策略「趋势展开」假设冲突。

### 11.2 止损放宽悖论

```text
threshold = max(8%, 2×ATR%)
```

波动越大止损越宽 → 急跌股实际亏损常 **>10%** 才离场，与「8% 硬止损」直觉相反。

### 11.3 盈利保护盲区

| 浮盈区间 | 有效保护 |
|----------|----------|
| < 0% | ① 成本止损 + ③④ 技术卖出 |
| 0% ~ 50% | ③ 通道 + ④ 双死叉（易洗出） |
| > 50% | ② 高点回撤 8%/12% |

**20%~50% 浮盈段无专用 trailing**，依赖通道/死叉，震荡市易回吐。

### 11.4 检查频率不对称

| 规则 | 10:00 | 14:30 | 盘后 |
|------|-------|-------|------|
| 亏损止损 | ✓ | ✓ | — |
| 盈利回撤 | ✓ | ✓ | 更新高点 |
| 通道 3 日 | ✓（执行） | — | 计数 |
| 双死叉 | ✓ | — | — |

盘中急跌：仅止损/盈利回撤可 14:30 响应；通道与死叉须等次日 10:00。

### 11.5 组合风控不减仓

大盘分从 80 降至 30 时，**存量持仓不会被动降低**；仅停止买入 + 弱市 6% 止损。2022 年实证：卖出 70 笔亏损，通道+死叉占主导。
