# trade2at4 详细策略说明 — 止损体系改造

> 对应代码：`trade2at4`  
> 平台：聚宽（JoinQuant）  
> 基准：沪深300（000300.XSHG）  
> 基线：`trade2`（仅改造 **止损体系**）

---

## 1. 策略定位

### 1.1 一句话

保留 trade2 全部选股与退出逻辑，将 **固定 -8% 单次检查** 改为 **动态阈值 + 双时点复检**，降低跳空滑点与「高波动股被误杀 / 低波动股止损过宽」问题。

### 1.2 相对 trade2 的唯一改动

| 项目 | trade2 | trade2at4 |
|------|--------|-----------|
| 止损阈值 | 固定 8% | **max(8%, 2×ATR14%)**，熊市收紧 6% |
| 检查时点 | 仅 10:00 | **10:00 + 14:30** |
| 通道 / 双死叉 / 买入 | 原样 | 原样 |

---

## 2. 动态止损阈值

### 2.1 计算公式

```python
threshold = g.hard_stop                    # 基础 8%
if market_score < 40:
    threshold = min(threshold, 0.06)       # 大盘弱时收紧至 6%

atr_pct = ATR(14) / 收盘价
threshold = max(threshold, 2.0 * atr_pct)  # 高波动放宽
```

**触发条件：** `浮盈率 ≤ -threshold` → 卖出。

### 2.2 参数

| 参数 | 默认值 | 说明 |
|------|--------|------|
| `g.hard_stop` | 0.08 | 基础止损 |
| `g.hard_stop_tight` | 0.06 | 弱市止损 |
| `g.hard_stop_score_threshold` | 40 | 低于此评分用 tight |
| `g.atr_stop_mult` | 2.0 | ATR 倍数 |

## 3. 双时点止损复检

```
07:30  盘前选股
10:00  market_open_trade   → 完整交易（含 10:00 止损）
14:30  afternoon_stop_check → 仅止损复检
15:00  盘后统计
```

14:30 只执行 `_check_stop_loss`，不买入、不检查通道/双死叉。
